# AI Monitoring Agent Dashboard

A modern React-based dashboard to monitor AI agents with real-time metrics, interactive charts, dark mode support, and control actions.

##  Features
-  Real-time CPU & Memory monitoring (auto-refresh)
-  Multiple AI agents (Vision AI, NLP Agent)
-  Start / Stop agent controls
-  Interactive charts using Recharts
-  Dark mode toggle
-  Sidebar navigation (Dashboard, Agents, Settings, Logs-ready)
-  Built with modern React (Vite)

## Tech Stack
- React 19
- React Router
- Recharts
- Tailwind CSS
- Vite
